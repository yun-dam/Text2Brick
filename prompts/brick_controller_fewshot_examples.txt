Below are few-shot Text-to-SPARQL samples for the Brick dataset.

Sample 1 (Latest Value)
Question: What is the latest reading for FCU fan speed?
Answer (SPARQL):
PREFIX brick: <https://brickschema.org/schema/Brick#>
PREFIX bldg: <bldg-59#>
PREFIX ref: <https://brickschema.org/schema/Brick/ref#>
PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>

SELECT ?timestamp ?value
WHERE {
  bldg:FCU_SPD ref:hasObservation ?obs .
  ?obs ref:hasTimestamp ?timestamp .
  ?obs ref:hasValue ?value .
}
ORDER BY DESC(?timestamp)
LIMIT 1

Sample 5 (Multi-Sensor Snapshot, modes)
Question: Show me the FCU control mode and fan operation mode for the last 10 readings
Answer (SPARQL):
PREFIX brick: <https://brickschema.org/schema/Brick#>
PREFIX bldg: <bldg-59#>
PREFIX ref: <https://brickschema.org/schema/Brick/ref#>
PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>

SELECT ?timestamp ?sensor ?value
WHERE {
  {
    SELECT ?timestamp ?value ("FCU_CTRL" AS ?sensor)
    WHERE {
      bldg:FCU_CTRL ref:hasObservation ?obs .
      ?obs ref:hasTimestamp ?timestamp .
      ?obs ref:hasValue ?value .
    }
    ORDER BY DESC(?timestamp)
    LIMIT 10
  }
  UNION
  {
    SELECT ?timestamp ?value ("FAN_CTRL" AS ?sensor)
    WHERE {
      bldg:FAN_CTRL ref:hasObservation ?obs .
      ?obs ref:hasTimestamp ?timestamp .
      ?obs ref:hasValue ?value .
    }
    ORDER BY DESC(?timestamp)
    LIMIT 10
  }
}
ORDER BY DESC(?timestamp)

Sample 6 (Multi-sensor at specific time)
Question: What were the cooling coil entering and return water temperatures on December 20th at 4 pm?
Answer (SPARQL):
PREFIX brick: <https://brickschema.org/schema/Brick#>
PREFIX bldg: <bldg-59#>
PREFIX ref: <https://brickschema.org/schema/Brick/ref#>
PREFIX xsd: <http://www.w3.org/2001/XMLSchema#>

SELECT ?timestamp ?sensor ?value
WHERE {
  {
    bldg:FCU_CLG_EWT ref:hasObservation ?obs .
    ?obs ref:hasTimestamp ?timestamp .
    ?obs ref:hasValue ?value .
    BIND("FCU_CLG_EWT" AS ?sensor)
    FILTER(?timestamp >= "2018-12-20T16:00:00"^^xsd:dateTime && ?timestamp < "2018-12-20T16:01:00"^^xsd:dateTime)
  }
  UNION
  {
    bldg:FCU_CLG_RWT ref:hasObservation ?obs .
    ?obs ref:hasTimestamp ?timestamp .
    ?obs ref:hasValue ?value .
    BIND("FCU_CLG_RWT" AS ?sensor)
    FILTER(?timestamp >= "2018-12-20T16:00:00"^^xsd:dateTime && ?timestamp < "2018-12-20T16:01:00"^^xsd:dateTime)
  }
}
ORDER BY ?timestamp
